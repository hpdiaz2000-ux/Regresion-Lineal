# Análisis Predictivo: Regresión Lineal

Este proyecto forma parte de mi portafolio profesional como **Ingeniero en Ejecución en Informática**, enfocado en el área de **Data Analytics & Business Intelligence**. Aquí se documenta el desarrollo de un modelo de aprendizaje supervisado para la predicción de variables continuas.

## 🚀 Descripción del Proyecto
El objetivo principal es implementar un modelo de Regresión Lineal capaz de predecir resultados basados en un conjunto de datos histórico. Se aplicaron técnicas de limpieza, análisis exploratorio de datos (EDA) y evaluación de métricas de desempeño.

## 🛠️ Tecnologías y Librerías
* **Python** (Lenguaje principal)
* **Pandas & NumPy** (Manipulación de datos)
* **Scikit-Learn** (Modelado y Machine Learning)
* **Matplotlib & Seaborn** (Visualización de datos)
* **Google Colab** (Entorno de desarrollo)

## 📈 Flujo de Trabajo
1. **Carga y Limpieza:** Tratamiento de valores nulos y normalización de variables.
2. **Análisis Exploratorio (EDA):** Visualización de tendencias y correlaciones.
3. **Entrenamiento:** División del dataset en conjuntos de entrenamiento y prueba.
4. **Evaluación:** Medición de desempeño mediante MAE, MSE y R-cuadrado.

## 📊 Resultados
El modelo final demostró una alta precisión en las predicciones, optimizando los parámetros para mejorar la eficiencia del análisis.
